MATCHUP TAB

This is the main screen. Use it to see what should be played right now.

WHAT YOU SEE HERE

  1. Current matchup: the red team, blue team, each team's skill total, and the difference.
  2. MID: the matchup ID. The same player combination keeps the same MID.
  3. Timer: shows how long the current matchup stays active.
  4. Session Maps: shows the maps planned for the session.
  5. Match Maker line: shows the match maker rotation, with the current person highlighted.

WHAT TO DO

  1. If a matchup is showing, read the teams and play.
  2. If there is no matchup, click CLICK TO GET STARTED.
  3. The web app will take you to the Generator tab so a matchup can be made.

BUTTONS AND FUNCTIONS

CLICK TO GET STARTED: appears only when there is no current matchup. It opens the Generator tab.

Session Maps card: this is view-only on the Matchup tab. To change maps, go to the Map List tab and unlock admin mode.

Simple rule: this tab is for viewing what everyone should play right now.

GENERATOR TAB

This tab makes balanced team options from the players you select.

ORDER TO USE IT

  1. Pick the Match Maker from the dropdown.
  2. Make sure the players who are playing are checked.
  3. Click CLICK TO GENERATE MATCHUP.
  4. Look through the generated matchup options.
  5. Click CLICK TO SELECT on the matchup you want to use.
  6. After it saves, the web app takes you back to the Matchup tab.

BUTTONS AND FUNCTIONS

Match Maker dropdown: choose who is making teams. Only players marked as Match Maker eligible appear here.

Player checkboxes: check the players who are playing. Uncheck anyone who is absent.

Sort by Name / Level: changes how the player checklist is sorted. It does not change player skill.

CLICK TO GENERATE MATCHUP: creates the matchup options.

Difference filters: show only matchups with that skill difference. Example: Difference 1 only shows matchups where the teams are 1 point apart.

Blitz recommendation: appears for uneven player counts. It can show matchups where the smaller team has the higher skill total.

CLICK TO SELECT: saves that matchup as the current matchup. This is the important final step.

If you generate matchups and try to leave without selecting one, the web app may warn you so you do not forget.

Simple rule: generate options, then always select the one you want.

HISTORY TAB

This tab shows old matchups that were selected.

WHAT YOU SEE HERE

  1. Date and time of the saved matchup.
  2. MID, Match Maker, how many times that matchup was picked, and the skill difference.
  3. The match card shows both teams, skill totals, difference, and maps played for that session.

ORDER TO USE IT

  1. Open the History tab.
  2. Click a history row to expand it.
  3. Read the teams and maps played.
  4. Click the row again to close it.

BUTTONS AND FUNCTIONS

LOAD ALL HISTORY: loads older history beyond the recent history shown first. Use this only when you need old records.

CLEAR MATCHUP HISTORY: admin-only. This erases the matchup history and resets the current matchup. Be careful with it.

Column headers: click headers like Date, MID, Match Maker, Times Picked, or Difference to sort the table.

History row: click a row to open the full match card.

The map edit controls inside an opened history card are only visible when admin mode is unlocked.

Edit maps played: admin-only. Use this if the wrong maps were saved or if a played map needs to be added, removed, or reordered.

Save map changes: admin-only. After editing maps inside a history card, save the changes before leaving.

Simple rule: history is where you check what happened before.

PLAYERS TAB

This tab is for managing players. The player list can be viewed, but changing players is protected by admin mode.

ORDER TO USE IT

  1. Unlock admin mode first.
  2. Click ADD PLAYER if you need a new player row.
  3. Edit names or skill numbers if needed.
  4. Click REMOVE only if that player should be removed from the active player list.
  5. Check or uncheck Match Maker if that player should or should not be allowed to make matchups.
  6. Click SAVE when done.

BUTTONS AND FUNCTIONS

ADD PLAYER: adds a blank player row.

Name cell: click a player name to edit it.

Skill controls: adjust a player's skill number. This affects team totals and generated matchups.

Match Maker checkbox: controls whether that player can appear in the Match Maker dropdowns.

REMOVE: removes that player from the active player list on this screen. The change is not final until you click SAVE.

SAVE: admin-only. Saves the player list to the sheet.

Changes not saved: means you edited something but have not clicked SAVE yet.

If admin mode is locked, player changes are not meant to be saved.

Simple rule: if you change players, click SAVE before leaving.

MAP LIST TAB

This tab controls the maps for the session.

WHAT YOU SEE HERE

  1. Session Maps: the maps planned for this session.
  2. Complete Map List: all maps the web app can use.
  3. Highlighted map: the last played map for that mode.
  4. Bonus: extra Elimination maps shown after CTF.

NORMAL SESSION ORDER

  1. Unlock admin mode before changing session maps.
  2. Click GENERATE SESSION MAPS to create the next map list.
  3. Play the maps.
  4. If a map was skipped, click the X next to that skipped map.
  5. When only played maps are still showing, click SAVE SESSION PROGRESS.
  6. The web app remembers what was played and brings skipped maps back later.

BUTTONS AND FUNCTIONS

Match Maker dropdown: admin-only. Chooses the current session's Match Maker. This also updates the Matchup tab.

X next to a map: admin-only. Removes a skipped map from the session list before saving progress.

SAVE SESSION PROGRESS: admin-only. Saves the maps still showing as played, then moves the map pointer forward.

GENERATE SESSION MAPS: admin-only. Creates the next map list from the saved map pointers.

CLEAR SESSION MAPS: admin-only. Clears the current session maps. Use this when you need to start fresh or prepare a custom session.

BUILD CUSTOM SESSION: admin-only. Lets you pick maps manually from the Complete Map List.

SAVE CUSTOM SESSION: admin-only. Saves the custom maps so they appear on the Matchup tab.

EXIT CUSTOM SESSION: admin-only. Leaves custom session mode and returns to the normal generated session setup.

COPY SESSION MAPS: copies the session map card as an image so it can be pasted somewhere else.

Bonus maps: Bonus is connected to Elimination. If Bonus maps are played, they count toward Elimination progress.

Simple rule: before saving progress, only maps that were played should still be showing.
